Data Visualization: Tic-Tac-Toe

View Depth: 5

Animation Speed: 3s

1) This is a game tree! The center point represents the "node" that's being displayed. All lines extending out represent connections to other nodes. You read the tree like a clock for available squares on the board. That means that the top branch going slightly right (below) represents X taking the top left square.The line going straight down (third line clockwise) represents X taking the 3rd available square, middle left. The branches extending farther follow the same pattern.

2) Whenever a node (some combination of X's and O's) results in a win for X (red), a win for O (blue), or a tie (green), they are depicted as a circle represented by that color.

3) Each node counts the number of X wins, O wins, and ties, then colors itself and the line to it
